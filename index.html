How does it work?

Simply enter the url of the target website and a keyword, and let L0k1 do the rest of the work.

Which filetypes does L0k1 search for?

By default, L0k1 will open a new tab with the entered sitename and keyword for: .PDF, .DOCX, .DOC, .CSV, .XLSX, .TXT, .RTF, .ODT, .PPTX, .PPTM, .PPT, .XML and .KML files.

By selecting Databases or Software under the More filetypes tab, it will search for: .PHP, .SQL, .SQLITE, .PDB, .IDB, .CDB, .SIS, .ODB, and, .ENV, .CFG, .CONF, .CONFIG, .CFM, .LOG, and .INF files respectively.

Browser compatibility

L0k1 currently works with Safari, Google Chrome and Mozilla Firefox browsers.

Note:

Chrome and Firefox users should allow L0k1 to send pop-ups and use redirects under the settings menu.

For Chrome users:

Settings > Security and privacy > Pop-ups and redirects > add L0k1 to sites allowed to send pop-ups and use redirects.

For Firefox users:

Privacy & Security > Block pop-up windows > Exceptions > add L0k1 to the list of sites that are allowed to send redirects
